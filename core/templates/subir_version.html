Cuando crear una nueva version

Usa este flujo cuando el archivo cambia pero sigue siendo el mismo documento de referencia.

  • Actualizacion de contenido manteniendo el mismo objetivo.
  • Nuevo respaldo que reemplaza al archivo anterior.
