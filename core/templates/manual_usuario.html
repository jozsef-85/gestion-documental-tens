Ayuda

Manual de uso de la aplicación

Guía rápida para trabajar con clientes, personal, presupuestos, seguimiento y documentos sin perder el orden operativo.

Ruta recomendada para empezar

Si es tu primera vez, sigue este orden para entender el flujo completo del sistema.
1. Clientes 2. Personal 3. Presupuestos 4. Seguimiento 5. Documentos

Cómo usar cada módulo

1
Clientes
Registra y consulta el padrón comercial. Aquí puedes crear, editar o eliminar del listado activo sin borrar historial.
2
Personal
Mantén fichas del equipo, respaldos y disponibilidad para trabajos activos.
3
Presupuestos
Usa la vista comercial para revisar cotizaciones, aceptación del cliente y monto antes de pasar a operación.
4
Seguimiento y documentos
Cuando el presupuesto ya fue aceptado, gestiona el trabajo, asigna personal, revisa facturación, pago y respalda documentos.

Tareas frecuentes

A
Agregar un trabajador a un presupuesto aceptado
Ve a Operación > Seguimiento, busca el presupuesto, entra en Agregar trabajador o Historial y usa el bloque Vincular trabajador.
B
Subir una planilla de presupuestos
Desde Operación > Importar planilla carga el archivo Excel. El sistema interpreta fecha, presupuesto, monto, nota de pedido, factura y pago.
C
Consultar documentos del trabajo
Abre el historial del presupuesto o usa el módulo Documentos filtrando por código de presupuesto.
D
Eliminar sin perder historial
En clientes y personal la eliminación es lógica: se quita del listado activo, pero se conserva el historial y las relaciones previas.

Buenas prácticas de uso

Trabaja con filtros
Usa búsqueda, estado, cliente y carga para encontrar resultados sin recorrer listados completos.
Mantén respaldo documental
Cada trabajo importante debería quedar vinculado a informes, checklists, fotos o certificados.
Cuida el estado real
El seguimiento cambia automáticamente según nota de pedido, factura y pago. Usa estado manual solo si necesitas corregir.
Cierra sesión al terminar
La plataforma guarda trazabilidad. Cerrar sesión ayuda a proteger la operación y el acceso a documentos.
